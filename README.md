# Portfolio-Website

## Summary
This project is to serve as the basis for my own portfolio website to showcase my Game Programming and Development knowledge and skills while working within solo and collaborative projects.

The overall structure of the website will include a home page that will give a quick summary about myself as a programmer and developer then scroll down to find some highlight projects (that have links to the portfolio page). I will also include a link to my resume (in pdf format) towards the bottom of the home page and also at the top with my LinkedIn and Github link. Then the about page will be going into more detail about my experiences and skills. Finally, the portfolio page will include all the projects that I have been apart of either as a solo developer or within a team environment.

## Flow
From the home page, viewers will be able to access the about and portfolio page with tab links at the top of the page. Also, from the highlighted project section on the home page, the images will include a link that brings them to the project page.
